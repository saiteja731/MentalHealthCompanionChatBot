# Intelligent Mental Health Companion Chatbot

An AI-powered chatbot that provides real-time emotional support and mental health assistance using Natural Language Processing and Deep Learning.

## Features
- Real-time emotional support using NLP
- Mood tracking and sentiment analysis
- Personalized responses based on user emotions
- Privacy and confidentiality focused

## Technologies Used
- Python
- Flask
- Keras/TensorFlow
- NLTK
- NumPy, Pandas, Scikit-learn

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python app.py
```
Then access the chatbot at `http://localhost:5006`
